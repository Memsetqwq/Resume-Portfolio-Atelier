# 真实素材补充引导

> 履历裁缝 Agent 想要做得更有人味,光靠公开设计资源不够。
> 这份文件告诉你**哪些真实素材最有价值**,以及怎么往这里放。

---

## 为什么需要真实素材

履历裁缝 = "8 年独立工作室主理人"。**只读公开设计理论的 Agent,跟只读教科书的医生一样,理论满分但开不出有针对性的方子。**

你提供真实素材(你自己的简历/朋友案例/看到好的简历/踩过的坑),Agent 在 Onboarding 时才能:
- 反推用户的真实审美(比直接问"你想要什么"准)
- 引用你见过的真实场景(不是凭空举"我有个客户...")
- 避开你踩过的雷(不说你已经知道会错的话)

---

## 应该放哪些素材(按价值排序)

### Tier 1 — 高价值(放进来立刻有用)

#### 1. 你自己的简历历史

把你之前发过的简历版本(无论好坏)放进来。**重点不是好的版本,是你踩过坑、改过不满意的地方**。

命名建议:`my-resume-v1.pdf` `my-resume-v2.pdf` `my-resume-v3-删了这段.md`

文件名加上"为什么改"最值钱(`v3-删了这段` `v4-加了项目截图` `v5-朋友吐槽了配色`)。

#### 2. 你看到好的简历/作品集

Dribbble / Behance / 即刻 / 知乎 / 公众号 / LinkedIn 截图。**标注一下"好在哪"**——比如"这个配色我喜欢" / "这个留白很舒服" / "这种大标题感觉很高级"。

命名建议:`good-001-vercel-style.png` `good-002-改用大图.jpg`

#### 3. 你踩过的坑

任何时候你做简历/作品集发现"这样做不行"的事:
- "投出去没回音,可能因为 X"
- "面试官吐槽 X 这段没讲清楚"
- "作品集大图在手机看不全"
- "投外企 ATS 没读出来"

命名建议:`pitfall-ats-没读html.md` `pitfall-设计师说我配色土.md`

### Tier 2 — 中价值(放进来更人味)

#### 4. 你朋友的简历/作品集(脱敏后)

朋友让你帮改过简历的,改完前 vs 改完后对比。**只放改动逻辑,不暴露朋友名字**。

命名建议:`friend-before-after/案例1-前端-配色太暗.md`

#### 5. 你行业的隐性规则

你所在的行业有些"潜规则"是公开资料查不到的:
- 投字节跳动要重点写"项目结果"而不是"用了什么技术"
- 设计岗作品集要"首屏就是最好的作品"
- 留学 CV 不要放照片,求职简历看国家

命名建议:`industry-rule-大厂前端.md` `industry-rule-设计师作品集.md`

### Tier 3 — 低价值(放进来只是参考)

- 公开的设计理论文章截图
- 通用简历模板截图
- 公开的优秀 GitHub README

这些 Agent 已经能查到,优先级低。

---

## 不应该放什么

- **任何含个人敏感信息的真实简历**(身份证/详细住址/未脱敏电话) —— Agent 没法保证这份本地文件不被误传到外部,你要保护自己
- **别人的真实简历**(哪怕是朋友)未经允许 —— 这是隐私
- **大文件**(每张图 > 5MB 的作品集截图) —— Agent 读起来慢,压缩到 < 1MB 再放
- **链接本身**(Agent 没法实时访问外链,只能读你下载下来的内容)

```

    │   ├── v2-投字节.md

    │   ├── good-001-vercel.png

```

## Agent 怎么读这些素材

Onboarding 阶段,Agent 看到你放了 Tier 1 的素材,会主动用——比如:
- "我看到你之前 v3 是这种风格,这次你想换吗"
- "你之前踩过的 X 坑,我现在写的时候怎么避开"

跨会话时,如果素材有更新,Agent 会自己读一遍最新内容。

---

## 隐私边界

- 这些素材**只在这个 Qclaw workspace 内**,不会上传到外部服务器
- 跟你聊过的 Agent 是 AI,不是真人,但**这些素材对 Agent 来说是它"看到"过的事实**
- 你随时可以让 Agent "忘掉某份素材",它会从记忆里删,但本地文件还是在你磁盘上(你要自己删)
- 不要把别人的真实信息(尤其是你不知道对方是否同意的)放进来——Agent 不会主动验证这一点,你自己把关

---

## 起步建议(没素材也能跑)

如果你现在手上没素材,Agent 也能正常工作——它会:
- 走"大厂风格推荐"流程,让用户自己挑
- 引用公开设计理论(虽然人味会淡一点)
- 用 Onboarding 主动追问的方式补齐信息

但 Agent 会更有人味**如果**你至少放了 1-2 份"自己简历历史 + 1 个踩坑"。从最小的开始,慢慢补。
